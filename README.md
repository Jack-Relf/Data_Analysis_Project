In this project, I estimated the parameter by fitting a theoretical model to experimental pion scattering data. I first defined a discrepancy function based on the weighted sum of squared residuals, where each residual was the difference between measured and predicted values, scaled by the experimental uncertainties. By varying w, I found the value that minimised this discrepancy, giving an initial best-fit estimate.

I then used SciPy’s curve_fit function to automate the fitting process. This method optimises the parameter more efficiently and returns a covariance matrix, allowing me to calculate the uncertainty in 𝑤 from the variance of the fit.

Finally, I implemented a Monte Carlo approach by simulating many datasets using Gaussian sampling based on the measured values and their uncertainties. I fitted the model to each simulated dataset, producing a distribution of w values. The mean of this distribution gave a robust estimate of w, while the standard deviation provided its uncertainty.
